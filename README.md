# ☁️ WeatherApp  

WeatherApp is an Android application that provides real-time weather information using the **OpenWeatherMap API**. Users can search for weather forecasts by city and view details such as temperature, humidity, and wind speed.

## 🚀 Features  
✅ Search for weather forecasts by city  
✅ Display current weather conditions (temperature, humidity, wind speed)  
✅ 5-day weather forecast  
✅ User-friendly interface  
✅ Material Design support  

##  Technologies Used  
- **Kotlin** - Primary programming language  
- **Retrofit** - API communication  
- **Gson** - JSON parsing  
- **ViewModel & LiveData** - Jetpack components  
- **RecyclerView** - List UI
